# ShopOpedia

ShopOpedia is a modern e-commerce application built with Vue 3, Pinia, Vue Router, and Firebase.

## Features

* Authentication & Authorization
* Protected Routes
* Product Catalog
* State Management with Pinia
* Responsive Design
* Custom 404 & 403 Pages

## Tech Stack

* Vue 3
* Pinia
* Vue Router
* Firebase
* Bootstrap 5
* Vite

## Live Demo

https://shop-opedia-rosy.vercel.app/

## Installation

```bash
npm install
npm run dev
```
